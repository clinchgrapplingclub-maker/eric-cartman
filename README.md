# 🎫 Discord Ticket Bot

A simple but powerful ticket bot built with discord.py.

## ✨ Features
- Slash command `/setup` (admin only)
- Custom ticket panel (title, description, images, color)
- Up to 3 ticket types (dropdown)
- Select category directly (no ID needed)
- Support team role (auto ping)
- Claim / Close / Delete ticket buttons
- Ticket transcripts sent to log channel
- `/remind` command (DM user to reply)
- Persistent ticket system
- Footer always: **made by @fntsheetz**
